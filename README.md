# go-pos-hardfork
Go Blockchain Hardfork Project (PoW -> PoS)


1. clone repo
2. go to ./run_nodes
3. run this command
```
./run_nodes.sh
```
4. open http://localhost:4000/
5. enjoy all pages
6. Blocks will be added to the chain every 5 seconds. Check http://localhost:4000/blocks)
7. kill all nodes using this command
```
./kill_nodes.sh
```
